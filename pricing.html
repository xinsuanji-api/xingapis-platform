常见问题
如何计费？
按 Token 用量计费。每次 API 调用会在响应中返回 input_tokens 和 output_tokens，按对应模型的单价计算费用，实时从账户余额扣除。
体验金有效期多久？
注册赠送的 ¥10 体验金有效期为 90 天，过期自动失效。充值金额无有效期限制。
支持哪些充值方式？
目前支持支付宝和微信支付，企业用户可申请对公转账。最低充值金额为 ¥10。
如何获取发票？
在控制台 → 充值记录 → 申请发票，填写发票信息后，我们会在 3 个工作日内开具电子发票并发送至您的邮箱。
余额不足会怎样？
余额不足时 API 请求会返回 HTTP 402（Payment Required），提示 "Insufficient balance"。请及时充值以免影响业务。
