3
发起第一次 API 调用

方式一：curl（最快测试）

# 替换 YOUR_API_KEY 为您的实际 Key
curl https://api.xinsuanji.cn/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "deepseek-v3",
    "messages": [{"role": "user", "content": "你好，介绍一下自己"}]
  }'

方式二：Python

from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",          # 您的星算纪 API Key
    base_url="https://api.xinsuanji.cn/v1"
)

response = client.chat.completions.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "解释量子计算"}]
)

print(response.choices[0].message.content)
print(f"用量: {response.usage.total_tokens} tokens")
print(f"费用: ¥{response.usage.total_tokens * 0.00014 / 1000:.6f}")

方式三：Node.js

import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.xinsuanji.cn/v1',

console.log(response.choices[0].message.content);

查看余额和调用日志

登录 控制台 可以实时查看：

  • 账户余额：当前剩余金额（¥）
  • API Keys：所有 Key 列表，可单独删除或禁用

5
Python / Node.js SDK 接入（可选）

Python 安装

pip install openai

Python 完整示例（多轮对话 + 流式输出）

from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.xinsuanji.cn/v1"
)

# 流式输出
stream = client.chat.completions.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "用一句话解释人工智能"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)

Node.js 安装

npm install openai

Node.js 完整示例

import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: process.env.XINSJ_API_KEY,
  baseURL: 'https://api.xinsuanji.cn/v1',
});

const models = await openai.models.list();
console.log(models.data.map(m => m.id));

常见问题 FAQ

Q：我的 API Key 泄露了怎么办？

立即登录控制台，在 API Keys 页面删除该 Key，然后重新创建一个新的。Key 删除后无法恢复，请谨慎操作。

Q：为什么调用返回 401 错误？

通常是 API Key 错误或未在请求头中正确设置 Authorization。请检查 Key 是否复制完整（以 sk- 开头，约 50 位）。

Q：为什么调用返回 402 错误？

账户余额不足。请在控制台充值，最低充值 ¥10。

Q：支持流式输出（Stream）吗？

支持！在请求体中设置 "stream": true（OpenAI SDK 默认启用流式）。

Q：如何切换模型？

修改请求体中的 "model" 字段即可，例如 "model": "gpt-4o" 或 "model": "claude-3.5-sonnet"。

Q：调用日志保留多久？

免费版保留 7 天，基础版（按量计费）保留 7 天，专业版保留 30 天，企业版永久保留。
