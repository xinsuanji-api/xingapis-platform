# 企业级 SLA（服务等级协议）

## 概述
星算纪 API 平台为企业客户提供 **99.99% 可用性 SLA**，确保您的业务持续稳定运行。

## SLA 等级

### 标准 SLA（所有客户）
- **可用性承诺**：99.9% 月度正常运行时间
- **每月不可用时间**：≤ 43.2 分钟
- **赔偿方案**：无（仅承诺）

### 企业 SLA（企业版客户）
- **可用性承诺**：99.99% 月度正常运行时间
- **每月不可用时间**：≤ 4.32 分钟
- **赔偿方案**：按服务中断时间比例退还月度费用

### 企业增强 SLA（定制版客户）
- **可用性承诺**：99.999% 月度正常运行时间（"五个九"）
- **每月不可用时间**：≤ 25.9 秒
- **赔偿方案**：按比例退还 **2 倍** 月度费用

## 测量方法

### 可用性计算
```
可用性 = (总分钟数 - 不可用分钟数) / 总分钟数 × 100%
```

### 不可用时间定义
- API 返回 5xx 错误超过 5 分钟
- API 响应时间超过 10 秒超过 5 分钟
- 数据丢失或数据损坏

### 排除项（不算入不可用时间）
- 计划内维护（提前 7 天通知）
- 客户操作不当（如 API Key 泄露）
- 第三方服务故障（如支付宝、阿里云）
- 不可抗力（如自然灾害）

## 赔偿方案

### 企业 SLA 赔偿表
| 月度可用性 | 赔偿比例 |
|------------|----------|
| < 99.99% 但 ≥ 99.9% | 10% |
| < 99.9% 但 ≥ 99% | 25% |
| < 99% | 50% |

### 企业增强 SLA 赔偿表
| 月度可用性 | 赔偿比例 |
|------------|----------|
| < 99.999% 但 ≥ 99.99% | 20% |
| < 99.99% 但 ≥ 99.9% | 50% |
| < 99.9% | 100%（全额退款）|

### 赔偿流程
1. **自动检测**：我们的监控系统会在每月 1 日自动计算上月可用性
2. **通知**：如果未达到 SLA，我们会主动通知您
3. **申请**：您可以在 7 天内申请赔偿
4. **审核**：我们会在 3 个工作日内审核
5. **赔付**：审核通过后，7 个工作日内完成赔付（退还到您的账户余额或原支付渠道）
