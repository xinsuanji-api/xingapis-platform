API 文档

星算纪提供 OpenAI 兼容 API，一行代码即可切换。

🚀 快速开始

只需 3 步，即可开始调用：

1. 注册账户
在 控制台 注册并获取 API Key（格式：sk-xxxxxxxx）

2. 查看余额
新用户注册即送 ¥10 体验金，无需充值即可测试。

3. 发起请求
使用下面的示例代码，替换 YOUR_API_KEY 即可：

# 对话生成（OpenAI 兼容）
curl https://api.xinsuanji.cn/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "deepseek-v3",
    "messages": [{"role": "user", "content": "你好！"}]
  }'

🔐 认证方式

所有 API 请求需要在 Header 中携带 API Key：

Authorization: Bearer sk-your-api-key-here
💡 API Key 可在 控制台 → API Keys 页面创建和管理。请妥善保管，不要在前端代码中暴露。

💬 对话生成

与 AI 模型进行对话，支持多轮对话、流式输出。

POST https://api.xinsuanji.cn/v1/chat/completions
Content-Type: application/json
Authorization: Bearer YOUR_API_KEY

{
  "model": "deepseek-v3",
  "messages": [
    {"role": "system", "content": "你是一个有帮助的助手。"},
    {"role": "user", "content": "解释量子计算"}
  ],
  "temperature": 0.7,
  "max_tokens": 2000,
  "stream": false
}

请求参数

参数类型必需说明
modelstring✓模型 ID
messagesarray✓对话消息列表
temperaturefloat采样温度，0-2，默认 0.7
max_tokensint最大输出 Token 数
streambool是否流式输出，默认 false

⚠️ 错误处理

所有错误响应均遵循 OpenAI 格式：

{
  "error": {
    "message": "Insufficient balance",
    "type": "insufficient_balance",
    "code": "balance_insufficient"
  }
}

错误码说明

错误码说明HTTP 状态码
invalid_api_keyAPI Key 无效或已过期401
insufficient_balance账户余额不足402
rate_limit_exceeded超出限流配额429
model_not_found模型不存在或无权限404
invalid_request请求参数错误400
upstream_error上游服务商错误502

🚦 限流说明

为保障服务稳定，平台对 API 调用实施限流：

套餐QPS（每秒请求数）TPM（每分钟 Token）
免费版350,000
基础版10200,000
专业版501,000,000
企业版自定义自定义
⚠️ 超出限流会返回 HTTP 429，并在 Retry-After Header 中告知等待秒数。
