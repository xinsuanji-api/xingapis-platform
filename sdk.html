🐍 Python 快速开始

3 行代码，完成第一次调用

bash 
pip install xingapis-sdk
python 
from xingapis import XingAPI

# 初始化（用您的 API Key）
client = XingAPI(api_key="sk-xxxxxxxxxxxx")

# 发起对话
response = client.chat.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "解释量子计算"}]
)

print(response.choices[0].message.content)
print(f"用量：{response.usage.total_tokens} tokens")
print(f"费用：¥{response.cost:.6f}")
python · 流式输出 
stream = client.chat.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "写一首诗"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
🟢 Node.js 快速开始

兼容 OpenAI SDK，改个 import 即可迁移

bash 
npm install @xingapis/sdk
# 或
yarn add @xingapis/sdk
typescript 
import { XingAPI } from '@xingapis/sdk';

const client = new XingAPI({
  apiKey: process.env.XING_API_KEY,
});

const response = await client.chat.create({
  model: 'deepseek-v3',
  messages: [{ role: 'user', content: '你好！' }],
});

console.log(response.choices[0].message.content);
✨ SDK 核心功能

所有 SDK 均支持以下能力

🔄
自动重试
遇到 429/5xx 错误自动指数退避重试，默认最多 3 次，可配置。
⚡
流式输出
原生支持 Server-Sent Events (SSE)，TypeScript/Python 均提供 async iterator 接口。
📊
用量统计
每次响应自动返回 token 用量和费用估算，无需手动计算。
🛡️
错误处理
统一错误类 XingAPIError，区分认证错误、余额不足、限流等场景。
🔌
中间件
Python/Node 均支持中间件机制，可注入日志、监控、审计等逻辑。
📦
批处理
支持批量 Embedding、批量对话请求，自动做并发控制，避免触发限流。
🔀 从 OpenAI SDK 迁移

只需改 2 行，其余代码 0 修改

python · 修改前（OpenAI 官方） 
from openai import OpenAI
client = OpenAI(api_key="sk-xxxx")  # ← 改这里
response = client.chat.completions.create(
    model="gpt-4o",                  # ← 改这里
    messages=[...]
)
python · 修改后（星算纪） 
from xingapis import XingAPI
client = XingAPI(api_key="sk-xxxx")   # ← 改成星算纪 Key
response = client.chat.create(
    model="deepseek-v3",               # ← 改模型名
    messages=[...]
)
